Takarazuka Information Suite v0.1.0
宝塚情報取得アプリ
======================================

■ 概要
宝塚歌劇公式サイトからニュース・公演案内を自動取得し、
見やすく一覧化するデスクトップ向けローカルHTMLアプリです。

■ 収集内容
  ・公式ニュース（全体 / 公演 / スター / 商品 / 配信・放送 / お知らせ）
  ・公演案内（宝塚大劇場・東京宝塚劇場 / その他の劇場）

■ 画面構成
  ・メイン            全件一覧 / 検索 / 絞り込み
  ・注目情報          新着・優先度上位の表示
  ・テーマ別          カテゴリ別に整理
  ・組別              花組 / 月組 / 雪組 / 星組 / 宙組 / 専科 で整理
  ・公演スケジュール  公演案内を一覧表示
  ・分析              件数集計

■ 使い方
  start_takarazuka_information.bat をダブルクリック

■ 必要ライブラリ（自動インストール）
  requests, beautifulsoup4

■ 取得元
  宝塚歌劇公式ホームページ
  - ニュース
  - 公演案内
